Text Image Reconstruction / Editorial Ink Issue 01

把参考图改写成一页有节奏的杂志式长图

这里放 1-2 句核心摘要。保留来源事实，但把原图的密集表达改成更像专题解读的阅读路径。

先建立判断，再展示证据；先让读者理解，再让读者复核。

Core Thesis
01 / Source Logic

先抽取事实，再重新组织叙事

把参考图中的产品名、数字、限制、例外和时间点列成事实清单。删除水印、平台口号和重复标题，保留会影响判断的信息。

01

核心结论

一句话说明这张图真正帮助读者判断什么。

02

关键证据

用短句重写来源数字、条件和差异，避免长串斜杠。

03

使用边界

把限制、待确认和免责声明放在读者能看到的位置。

02 / Evidence

每个模块只承担一个观点

如果来源图很密，拆成几个专题模块。每个模块有标题、证据和解释，不把全部信息塞进一个表格。

模块 A：变化点 放前后对比、升级点、核心数字或一条最重要的限制。
模块 B：适合谁 用来源信息支持决策，不加入没有证据的推荐。

结尾必须回到读者怎么用

本模板适合文章图、产品解读、保险/金融说明、政策变化拆解。交付前替换全部占位文字，确认所有数字与来源图一致，并运行 HTML 可读性校验。
